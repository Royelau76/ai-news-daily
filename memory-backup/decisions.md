## [D-001] 采用 3-File Pattern 重构记忆系统

**决策时间**: 2026-02-12 23:56
**决策人**: 瑞哥 + 小莫

### 决策内容
将原有的单一 MEMORY.md 管理模式，改为 3-file pattern:
- `tasks.md` - 任务管理
- `learnings.md` - 知识积累
- `memory/YYYY-MM-DD.md` - 每日进展（保持原有）

### 决策原因
1. **解决失忆问题**: 上下文压缩后会丢失信息，需要强制写文件
2. **分离关注点**: 任务、发现、进展分开管理，更清晰
3. **社区验证**: MyloreAgent 等 Agent 证实此方法有效

### 预期效果
- 任务追踪更清晰
- 知识可复用，避免重复踩坑
- 上下文恢复更快速

## [D-002] AI新闻日报采用静态 HTML + GitHub Pages

**决策时间**: 2026-02-12
**决策人**: 瑞哥

### 决策内容
使用 Python 生成静态 HTML，通过 GitHub Actions 自动部署到 GitHub Pages

### 备选方案
- ❌ Jekyll 主题方案（构建复杂，容易失败）
- ❌ 纯 Markdown（飞书预览体验差）
- ✅ 纯 HTML 静态文件（简单可靠）

### 决策原因
1. **简单可靠**: 不需要 Jekyll 构建，直接服务静态文件
2. **速度快**: HTML 直接渲染，无需转换
3. **可定制**: 完全自定义样式和布局

## [D-004] 为十一采用"小份量多餐+手喂"纠正挑食

**决策时间**: 2026-02-12
**决策人**: 瑞哥

### 决策内容
针对十一的护食+挑食问题，采用综合方案：
1. 分成 3-4 小份
2. 手喂建立信任
3. 漏食玩具增加趣味
4. 严格执行"不吃完不补"

### 决策原因
- 护食导致无法使用"15分钟收走"法则
- 贵宾犬聪明，需要心理战术
- 8岁进入中老年，需要规律饮食

### 预期效果
- 3-5 天内改善挑食
- 建立"人=食物来源"的信任关系
- 保持标准体重
