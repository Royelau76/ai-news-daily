# CONFIG.md - 配置与操作指南

## 🏠 工作空间规范

### 每次会话必做
1. 读 `SOUL.md` — 我是谁
2. 读 `USER.md` — 帮助谁
3. 读 `memory/YYYY-MM-DD.md`（今天+昨天）
4. **主会话**：额外读 `MEMORY.md`

### 记忆管理
| 类型 | 文件 | 用途 |
|------|------|------|
| 每日记忆 | `memory/YYYY-MM-DD.md` | 当天事件记录 |
| 长期记忆 | `MEMORY.md` | 筛选后的重要信息 |
| 专题记忆 | `memory/专题名.md` | 特定主题（如高考） |

**归档机制**：每周日将7天记忆合并为周总结，删除原始日文件
