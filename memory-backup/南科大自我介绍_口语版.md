# 南方科技大学综合评价自我介绍（口语版）

---

**尊敬的南科大招生老师：**

您好！我是一名来自云南省的高三学生，从小就喜欢捣鼓电脑，得知南科大是一所以创新著称的"双一流"高校，我非常向往，特此报名参加贵校的综合评价招生。

**关于南科大**

选择南科大，首先是被它的教育理念吸引——"明德求是、日新自强"，培养具有家国情怀和创新能力的人才，我觉得这正是新时代所需要的。我在网上了解到，南科大位于深圳，学术氛围开放国际化，而且特别注重学生的综合素质，不只看高考成绩，这让我觉得自己的特点有机会被看到。

我尤其对计算机和信息技术相关领域感兴趣。南科大计算机系在人工智能、网络空间安全等方向实力很强，深港微电子学院在集成电路领域也很有特色。大一、大二可以自主选择专业这个机制也很好，让我能先打好基础，再确定具体方向。

**个人经历**

说到和编程的缘分，大概从小学四年级就开始了。那时候学校开了信息技术课，我第一次接触到电脑编程就特别感兴趣，自己在网上找教程学。后来初中、高中一直坚持学习，偶尔会写一些小程序满足自己的需求，比如自动整理文件的小工具、学习用的背单词程序之类的，虽然都比较简单，但都是自己动手完成的，特别有成就感。

**关于竞赛**

高一的时候，我参加了云南省的NOIP（全国青少年信息学奥林匹克联赛），获得了省级一等奖。这次比赛让我更深入地理解了算法和数据结构，也让我明白编程不只是写代码，更是一种解决问题的思维方式。

**个人特点**

性格方面我比较开朗，喜欢和人交流，也愿意尝试新事物。学习上我比较认真，钻起问题来经常忘了时间。当然也有短板——体育实在是不太行（扶额），好在平时打打羽毛球、跑跑步，也能锻炼身体，保持身心愉悦。

**结语**

我真诚地希望能够进入南科大读书，在更好的平台上学习计算机专业知识，提升自己的综合能力。如果能被录取，我一定珍惜机会，好好学习，争取将来能用自己的技术为社会做点贡献。

感谢老师抽出宝贵时间阅读我的申请！

此致

敬礼！
