## 💡 OpenClaw使用技巧

### 技巧1：使用Cron定时任务自动生成日报
```yaml
# 在配置中添加定时任务
cron:
  jobs:
    - name: "AI新闻日报"
      schedule:
        kind: "cron"
        expr: "0 9 * * *"  # 每天早上9点
        tz: "Asia/Shanghai"
      payload:
        kind: "agentTurn"
        message: "执行AI新闻日报任务：搜集AI动态并生成日报..."
      sessionTarget: "isolated"
```

### 技巧2：使用设备配对功能控制手机
OpenClaw 2026.2.9新增设备配对插件，支持：
- 通过Telegram `/pair` 命令配对设备
- 远程控制iOS/Android节点
- 访问手机摄像头、位置、屏幕等功能

### 技巧3：配置多模型故障转移
```yaml
agents:
  defaults:
    models:
      - kimi-coding/k2p5      # 主模型
      - openai/gpt-4o         # 备选1
      - anthropic/claude-3.5-sonnet  # 备选2
    failover: true             # 启用自动故障转移
```

### 技巧4：使用xAI Grok作为搜索提供商
最新版本支持将Grok (xAI) 作为web_search提供商：
```yaml
tools:
  web_search:
    provider: "xai"  # 新增选项
```
